One-line installer (works on every Linux you'll ever SSH into).

{{ partial "code-block.html" (dict "code" $linuxInstall "lang" "sh" "id" "linux-install") }}

The script detects your OS and architecture, downloads the matching archive from the latest GitHub Release, and drops the spiceedit binary into ~/.local/bin (or /usr/local/bin if ~/.local/bin isn't writable). Re-run it to upgrade. Plain POSIX sh, no bash required — works on Alpine, BusyBox, and any SSH target.

{{/* Windows panel */}}

Windows builds (amd64) ship as a zipped binary on every GitHub Release. Download the archive, unzip spiceedit.exe, and drop it somewhere on your PATH. There is no installer or Chocolatey package yet — the binary works in Windows Terminal, ConEmu, and WSL.
